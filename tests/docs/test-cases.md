## TC-013 Проверка пополнения баланса

Уровень критичности: Критический
Предусловие: Пользователь должен быть зарегистрирован и авторизован

1. Кликнуть на кнопку 'Transactions'
2. Кликнуть на кнопку 'Add balance'
3. Ввести в поле 'Enter sum' нужную сумму для пополнения баланса

Ожидаемый результат: На балансе появится пополнение на указанную сумму, а под хедером 'Transactions' появится информация об этом пополнении
